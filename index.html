P5 Animations

A collection of animations made using P5.js, source code for this website can be found on github

Some animations are intensive to run, so mobile friendly ones are marked with the icon

Many of these projects have been inspired by the following sources and tutorials: The coding Train, Numberphile, 3Blue1Brown

Hexes

A page of tiled hexes that grow/shrink in a random or center-out pattern. Click on the hexes animation to make it full screen.

Beautiful Trigonometry

Points on the spokes of a circle following a main point as it rotates around the edge of the circle. Press the button to show how the points are calulated and how trigonometry is related to circles.

Flocking Simulation

A simulated flock of birds using Boids program.

Snowflake Generator

Repeatedly grow a random snowflake until it reaches full size and then begin again.

Modulo Times Table

A cicle with numbers 0 to N-1 around the outside. Each number is then multiplied by a factor F modulo N, and a line is drawn from it to the result. The multiplication factor is increased as time goes on.

Matrix Rain

The falling streams of symbols seen in the Matrix films.

Lissajous Curve Table

A table with rotating cogs in the top row and left column. The speed of the cogs increases going right/down along the table. Horizontal and vertical lines are then drawn from a point on the cogs edge. Where these lines intersect a path is traced.

Maurer Rose

A set of points along rose curve connected with straight lines. Animated so that the set of points and number of rose petals changes over time.

Chaos Game

Place a series of N points, 3 points in this example. Draw a dot at a random place on the screen. Randomly chose a point and move the dot halfway to it, and then repeat forever. Depending on the number of points, different patterns will appear.
